## JavaScript Integration

### Load Data
```javascript
let jobs = [];

async function loadJobs() {
  const response = await fetch('/data/jobs_v6_4cat.json');
  jobs = await response.json();
  console.log(`Loaded ${jobs.length} jobs`);
}
```

### Search Function
```javascript
function searchJobs(query) {
  const q = query.toLowerCase().trim();
  if (!q || q.length < 2) return [];
  
  const results = [];
  
  // Priority 1: Exact alias match
  for (const job of jobs) {
    const aliases = job.aliases || [];
    if (aliases.some(a => a.toLowerCase() === q)) {
      results.push({ ...job, matchType: 'alias_exact' });
    }
  }
  
  // Priority 2: Alias contains query
  for (const job of jobs) {
    if (results.some(r => r.id === job.id)) continue;
    const aliases = job.aliases || [];
    if (aliases.some(a => a.toLowerCase().includes(q))) {
      results.push({ ...job, matchType: 'alias_partial' });
    }
  }
  
  // Priority 3: Title contains query
  for (const job of jobs) {
    if (results.some(r => r.id === job.id)) continue;
    if (job.en.toLowerCase().includes(q) || job.de.toLowerCase().includes(q)) {
      results.push({ ...job, matchType: 'title' });
    }
  }
  
  return results.slice(0, 15);
}
```

  'Human-Centric': '#22c55e',      // green
  'AI-Assisted': '#3b82f6',         // blue

function JobProfile({ job }) {
  return (
    <div className="job-profile">

          </div>

          >
            AI-Assisted {job.ai_assisted}%
          </div>
        )}
        {job.agentic > 0 && (

      </div>

## Methodology Note (for About page)

```
This tool analyzes AI exposure using a 4-category framework:

1. Human-Centric: Tasks requiring human presence, judgment, or empathy
2. AI-Assisted: Complex work where AI tools help but humans lead
3. Agentic Potential: Tasks AI agents can handle with minimal oversight
4. High Automation: Routine tasks with high automation potential

The "Agentic Potential" category identifies work that emerging AI agents 
(like coding assistants, data pipelines, and workflow automation) can 
increasingly handle autonomously.

Data sources:
- German Federal Employment Agency (KldB 2010) for task definitions
- O*NET (US Department of Labor) for interpersonal skill requirements
```
